## Deployment

### Vercel Deployment

For Vercel deployment of the Pokemon Card Analyzer:

**Option 1: Set Root Directory (Recommended)**
1. Go to Vercel project settings
2. Set "Root Directory" to `pokemon-card-analyzer`
3. Deploy

**Option 2: Use Monorepo Setup**
1. Keep current structure
2. Configure Vercel to build from `pokemon-card-analyzer` directory
3. Update build settings as needed

### Local Development

Each project can be run independently:
- **Pokemon Card Analyzer**: Node.js app (see subdirectory README)
- **Beans Page**: Static HTML (open `beans.html` in browser)
